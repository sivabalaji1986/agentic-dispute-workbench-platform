# Session 1 — Architectural Hardening Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close four hardening gaps identified before Session 2 begins: defend `List<>` immutability in every affected record, enrich `CaseReviewResult`/`EvidenceReadiness` so the evidence-merge contract won't need a breaking change in Session 3, verify `agui-support` never triggers Spring Boot autoconfiguration, and add a `contract-tests` module with golden JSON fixtures generated from the real builders so serialization drift is caught immediately in every future session.

**Architecture:** This is a fix-and-harden pass over the two existing modules (`workbench-common`, `agui-support`) plus one new plain-Java test-only module (`contract-tests`). No new production module logic, no new packages beyond what's explicitly listed. `contract-tests` depends on both existing modules at test scope only and asserts their real builder/record output against frozen fixture JSON files.

**Tech Stack:** Same as Session 1 — Java 25, Maven multi-module, Jackson 3 (`tools.jackson.*` for core/databind, `com.fasterxml.jackson.annotation.*` for annotations), JUnit 5.

## Global Constraints

- This is a **targeted fix pass only**. Do not change method signatures, package structure, field names, or anything not explicitly listed in a task below. Do not add features beyond what each task specifies.
- `PLATFORM_CONTRACT.md` remains the source of truth; nothing in this plan changes it.
- Jackson package rule unchanged from Session 1: `tools.jackson.core`/`tools.jackson.databind` for ObjectMapper/core/databind; `com.fasterxml.jackson.annotation.*` for annotations (`@JsonProperty`, `@JsonInclude`) — this is the one permitted `com.fasterxml.jackson` import, verified in Session 1.
- Use `List.copyOf(...)` for defensive copies, never `Collections.unmodifiableList(...)` — `copyOf` makes an actual defensive copy; `unmodifiableList` is just a non-defensive wrapper over the same backing list.
- All fixture JSON in this plan was generated by actually running the real builders against a live build (not hand-typed) — see each task's fixture content for the exact, verified bytes.

```java
package com.workbench.common.merge;

import com.fasterxml.jackson.annotation.JsonInclude;
import com.workbench.common.agui.EvidenceItem;

import java.util.ArrayList;
import java.util.List;

@JsonInclude(JsonInclude.Include.NON_NULL)
public record EvidenceReadiness(int present, int required, List<String> missingItems, String label) {

    public EvidenceReadiness {
        missingItems = missingItems == null ? List.of() : List.copyOf(missingItems);
    }

    public static EvidenceReadiness compute(List<EvidenceItem> available, List<String> required) {
        List<String> missingItems = new ArrayList<>();
        int presentCount = 0;
        for (String requiredCode : required) {
            String humanReadable = DocumentTypes.humanReadable(requiredCode);
            boolean found = available.stream()
                    .anyMatch(item -> item.present() && item.label().equalsIgnoreCase(humanReadable));
            if (found) {
                presentCount++;
            } else {
                missingItems.add(humanReadable);
            }
        }
        String label = presentCount + " of " + required.size() + " required items present";
        return new EvidenceReadiness(presentCount, required.size(), missingItems, label);
    }
}
```

**Design note on `A2uiPayloadContractTest`:** `A2uiMessages`/`A2uiComponents` deliberately return `Object` backed by `private` nested records (see Session 1's `A2uiMessages.java`/`A2uiComponents.java`, and PLATFORM_CONTRACT.md §11) — there is no named, externally-accessible Java type to deserialize a fixture into for these. Making those records `public` just to satisfy a generic "deserialize into the corresponding type" description would be an unlisted signature/visibility change, which this plan's Global Constraints forbid. Instead, each `A2uiPayloadContractTest` method invokes the real builder, serializes its live output, and asserts it against the fixture both as an exact string AND as a parsed `JsonNode` tree (order-independent, defense in depth) — this still fails immediately on any field rename or shape drift in the builders, which is Fix 4's actual goal. `CommonDtoContractTest` and `AguiEventContractTest` cover types that ARE public (`CaseReviewResult`, `PolicyResult`, `EvidenceReadiness`, `AguiEvents.*`), so those deserialize into the real type as literally described.

```java
package com.workbench.contract;

import com.workbench.agui.a2ui.A2uiComponents;
import com.workbench.agui.a2ui.A2uiMessages;
import com.workbench.common.agui.ActionItem;
import com.workbench.common.agui.EvidenceItem;
import org.junit.jupiter.api.Test;
import tools.jackson.databind.JsonNode;
import tools.jackson.databind.ObjectMapper;
import tools.jackson.databind.json.JsonMapper;

import java.io.IOException;
import java.io.InputStream;
import java.nio.charset.StandardCharsets;
import java.util.List;

import static org.junit.jupiter.api.Assertions.assertEquals;

/**
 * A2uiMessages/A2uiComponents deliberately return {@code Object} backed by private nested
 * records (PLATFORM_CONTRACT.md §11) — there is no named, accessible Java type to deserialize
 * a fixture into. Each test instead invokes the real builder, serializes its live output, and
 * compares both the exact JSON string and the parsed JsonNode tree against the frozen fixture,
 * so a field rename or shape change in the builder fails here immediately.
 */
class A2uiPayloadContractTest {

    private final ObjectMapper objectMapper = JsonMapper.builder().build();

    @Test
    void createSurfaceMatchesFixture() throws IOException {
        String fixture = loadFixture("a2ui-create-surface.json");

        Object live = A2uiMessages.createSurface(
                "case-D-10291", "https://dispute-workbench.internal/catalogs/v1.json");
        String liveJson = objectMapper.writeValueAsString(live);

        assertEquals(fixture, liveJson);
        assertEquals(objectMapper.readTree(fixture), objectMapper.readTree(liveJson));
    }

    @Test
    void updateComponentsFullMatchesFixture() throws IOException {
        String fixture = loadFixture("a2ui-update-components-full.json");

        Object decisionCard = A2uiComponents.decisionCard(
                "decision-1", "checklist-1", "actions-1",
                "Needs More Evidence", "Goods Not Received",
                "2 of 4 required items present", "Create evidence request task");
        Object checklist = A2uiComponents.evidenceChecklist("checklist-1", List.of(
                new EvidenceItem("Transaction record", true),
                new EvidenceItem("Merchant response", true),
                new EvidenceItem("Customer declaration", false),
                new EvidenceItem("Delivery / non-delivery proof", false)));
        Object actions = A2uiComponents.nextActions("actions-1", List.of(
                new ActionItem("create_evidence_request_task", "Create Evidence Request Task"),
                new ActionItem("escalate_to_reviewer", "Escalate to Reviewer"),
                new ActionItem("save_case_note", "Save Case Note")));
        Object live = A2uiMessages.updateComponents("case-D-10291", List.of(decisionCard, checklist, actions));
        String liveJson = objectMapper.writeValueAsString(live);

        assertEquals(fixture, liveJson);
        assertEquals(objectMapper.readTree(fixture), objectMapper.readTree(liveJson));

        JsonNode decisionCardNode = objectMapper.readTree(liveJson)
                .get("updateComponents").get("components").get(0);
        assertEquals("checklist-1", decisionCardNode.get("checklistId").asString());
        assertEquals("actions-1", decisionCardNode.get("actionsId").asString());
    }

    @Test
    void approvalPreviewMatchesFixture() throws IOException {
        String fixture = loadFixture("a2ui-approval-preview.json");

        Object live = A2uiComponents.approvalPreview(
                "approval-1", "D-10291", "Pending Evidence",
                List.of("Customer declaration", "Delivery / non-delivery proof"),

        String liveJson = objectMapper.writeValueAsString(live);

        assertEquals(fixture, liveJson);
        assertEquals(objectMapper.readTree(fixture), objectMapper.readTree(liveJson));
```
